Features

Everything you need.
Nothing you don't.

Built from the ground up for rooted Android devices deep system access, real-time data, and fine-grained control.

Performance Profiles

Switch between Balance, Performance, Gaming, and Battery Saver profiles with a single tap. Applies CPU governor, GPU freq, and I/O tweaks automatically.

Kernel Tweaks

17+ individual tweaks across CPU, GPU, memory, I/O, network, and battery. Toggle each one independently with live validation.

System Monitor

Real-time stats for SoC, RAM, CPU/GPU frequency, temperature, and uptime all updated live without draining your battery.

App Profile

Per-app optimization with automatic CPU governor, refresh rate, and system tweak switching. Game differently, browse efficiently.

Magisk & KernelSU

First-class support for Magisk, KernelSU, and APatch. Automatic root manager detection with graceful fallback handling.

Bug Reports

Found an issue?

Only reports with full logs will be accepted. Include the right info based on issue type below.

Root / Permission
Root or permission issue

Include root manager name & version, Android version, and MIUI/OneUI/AOSP info.

App Crash
App crashes or force closes

Record and upload a full logcat captured at the moment of the crash. Use adb logcat or Matlog.

Tweak Issue
Tweak not working correctly

Include device model, SoC name (e.g. Snapdragon 778G), and a screenshot of the Log tab.

DWYOR
Do With Your Own Risk

Aether Manager performs modifications at the kernel and root system level. Make sure you fully understand each tweak before enabling it. The developer is not responsible for any device damage, bootloops, data loss, or other system issues that arise from using this application. Misuse or incorrect configuration is entirely at your own risk. Use wisely.
